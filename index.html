01 — ML

House Price Prediction using Regression Model

This project aims to build a machine learning model that predicts residential property prices based on key housing features. The goal is to assist buyers, sellers, and real estate analysts in making data-driven pricing decisions. The model was trained on a housing dataset containing both numerical and categorical features, including location (latitude and longitude), median income, total rooms, total bedrooms, population, households, and ocean proximity. The target variable is the median house value. A complete data preprocessing pipeline was implemented using ColumnTransformer and Pipeline, which handled feature scaling with StandardScaler and encoding of categorical variables. Exploratory Data Analysis (EDA) was performed using visualizations such as heatmaps and box plots to identify correlations and detect outliers. The initial model was built using Linear Regression, and performance was improved through feature selection and outlier handling techniques. The model was evaluated using R² Score and Mean Squared Error (MSE) to measure prediction accuracy. The final model achieved an R² score of approximately 0.81, demonstrating a reasonable ability to capture trends in housing prices. This project highlights practical experience in data preprocessing, model building, and evaluation using machine learning workflows.
